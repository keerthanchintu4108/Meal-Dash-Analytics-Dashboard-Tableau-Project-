# 🍽️ Meal Dash Analytics Dashboard (Tableau)

## 📌 Overview
This project showcases an interactive Tableau dashboard built using Meal Dash data. It provides insights into order trends, customer preferences, and delivery performance to enhance business strategies.

## 📊 Features
- Order and revenue analysis
- Customer behavior insights
- Delivery performance tracking
- Time-based trend analysis
- Interactive filters and dashboards

## 🛠️ Tools Used
- Tableau
- Microsoft Excel

## 📁 Dataset
The dataset includes information about orders, customers, delivery details, and revenue used for visualization and analysis.

## 🚀 How to Use
1. Download the `.twbx` Tableau file
2. Open using Tableau Desktop
3. Explore dashboards with interactive filters

## 🎯 Objective
To analyze food delivery data and generate actionable insights through interactive visualizations.

## 📌 Conclusion
This dashboard helps identify key trends and improve operational efficiency, customer satisfaction, and overall business performance.
